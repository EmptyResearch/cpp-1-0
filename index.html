Lesson 1.0: Output with cout

What is Cout?

    Cout is one of the most important functions in C++. Cout stands for "(C)onsole (OUT)put", and its function is pretty self-explanatory: it outputs to the console. All C++ compilers have a console, and all output with cout goes to the console.

Syntax of Cout

    The syntax of cout is easy to learn, and catches on quickly, but it can be difficult to remember which operator to use (you'll see later). Here is an example program with cout:

#include <iostream>
int main(){
  using namespace std;
  cout << "Hello World!";
}
    This is a famous program; the Hello World program. As you can see, the program's result will be:

Hello World!

(By the way, all strings in C++ have quotes (""). We'll learn more about this later when we discuss data types.)
     The syntax of cout is "cout << output;". You'll now see what I meant about mixing up the operators; some programmers accidentally type "cout >> output;". See the difference? The actual cout function uses << operators. The time when you need to use >> operators is cin. We'll discuss cin in the next lesson.

Summary

  • Use cout to output data of any type.
  • The syntax of cout is "cout << output;".
  • Remember to use the << (less than) operators instead of using the >> (greater than) operators.
