Lesson 1.0: Output with cout

What is Cout?

    Cout is one of the most important functions in C++. Cout stands for "(C)onsole (OUT)put", and its function is pretty self-explanatory: it outputs to the console. All C++ compilers have a console, and all output with cout goes to the console.

Syntax of Cout

    The syntax of cout is a bit of a surprise at first, especially if you're used to JavaScript or Python. Here is an example program with cout:

#include <iostream>
int main(){
  using namespace std;
  cout << "Hello World!";
}
